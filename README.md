# DLT645报文组装
读正向
```
Frame = #{
    <<"command">> => 17,
    <<"addr">> => <<65,7,0,0,22,4>>,
    <<"di">> => <<0,1,0,0>>
},
shuwa_smartmeter:to_frame(dlt645, Frame).

返回报文 -->
<<104,65,7,0,0,22,4,104,17,4,51,51,52,51,20,22>>

```
